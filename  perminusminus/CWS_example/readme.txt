全球各大媒体都在报道姚明已决定退役，姚明的经纪团队姚之队却一口否认，姚明到底决定退役了吗？我们应该相信谁？媒体vs姚之队，显然后者更权威。但现在情况发生了变化。

　　因为NBA官方也表态了。

　　北京时间7月9日，在美国各大媒体纷纷报道姚明决定退役并已通知火箭队和NBA联盟之后，NBA官方网站也在要闻头条刊登了姚明退役的消息。NBA官方电视台也报道了姚明决定退役的新闻。

　　>> 视频-NBA联盟官方网站消息 姚明通知联盟选择退役

　　>> 美媒体曝姚明决定退役！ NBA官方网站刊出消息确认

　　NBA官方网站称：根据NBA联盟内部消息，第一位闯入NBA的中国大明星姚明，在效力NBA九个赛季后已经决定退役。

　　NBA官网回顾了姚明NBA生涯的数据，并总结道：身高7英尺6英寸的姚明于2002年以新秀状元的身份进入NBA，在NBA在全球特别是亚洲地区的扩张期间，在字面上和实际上都成为NBA最大的标志。但在他的NBA生涯中，他的形象变得越来越脆弱。在前三个赛季只因伤缺席了两场比赛后，后六个赛季一共缺席了250场比赛。

　　文章称姚明虽然曾渴望再一次尝试复出，但现在他已经不想为此损害到未来的健康。姚明的父母甚至在2009年5月姚明在季后赛中脚骨骨折后就不希望他回到NBA赛场。姚明自己也在女儿出生后表示：“我希望以后能和我的孩子一起奔跑玩耍，而不希望永远要戴着护具、拄着拐杖。”

　　如果说姚之队章明基先生今天凌晨接受新浪体育采访时否认姚明已决定退役，还让我们为姚明复出心存一丝希望，NBA官方网站的这番报道则基本上把我们这最后的一丝希望也浇灭了。除非，姚明本人亲自出来澄清一切。

　　由于NBA停摆的关系，NBA官方网站的文字或视频报道中都没有出现姚明的形象。火箭队官方网站更是因为NBA停摆期间的诸多限制，连姚明的名字都不能提及。做为姚明效力了九个赛季并深爱的球队，此时此刻，在姚明告别之际，火箭队既不能向姚明九年来的贡献致谢，也不能向他表示祝福，难免令人唏嘘。

　　姚明自2002年登陆NBA以来，一直效力于火箭队，场均为火箭贡献了19分、9.3篮板和1.9盖帽。2008-2009赛季，打了77场比赛的姚明场均攻下19.7分、9.9篮板和1.9盖帽，成功率领火箭在季后赛首轮淘汰了开拓者，可惜在对湖人的第二轮比赛中受伤，就此开始了他漫长的养伤期。火箭的腾空进程也因此戛然而止，在姚明这次受伤之后，火箭队就再未能杀入过季后赛。

　　虽然姚明在NBA只打了9个赛季，但是他留下的荣誉却是无数。作为NBA历史上第一位外籍状元秀，除了整季伤停的2009-2010赛季，他连续8次入选全明星。5次入选联盟最佳阵容。加入联盟的第一年，就入选了最佳新秀第一阵容。在国际赛场上，姚明同样成就惊人，他率领中国男篮在2004年雅典奥运会上进入过前八，也在2006年日本男篮世锦赛上当选过赛事得分王。这些辉煌，足够让球迷永记姚明的伟大了。
